Key Decisions Made

  • Zero-friction onboarding — Users get a BSV address instantly, no wallet download
  • Server-funded posts — We pay for on-chain transactions (~$0.0005 each)
  • Simple key storage — WIF in localStorage with copy/download backup options
  • Anonymous identity — anon_XXXX names tied to BSV keypairs
  • Signed posts — Every post is cryptographically signed for proof of authorship
  • Minimal UI — ChatGPT-style interface to reduce friction
  • Dynamic imports — BSV SDK loaded only when needed to avoid bundling issues
  • Auto-migration — Old identities upgraded to include keypairs without data loss
